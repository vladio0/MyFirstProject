# MyFirstProject

Як запушити на GitHub за допомогою терміналу:

Terminal -> git init -> git status -> git add. -> git commit -m "Initial commit" -> переходимо в GitHub в браузері
-> new Repository -> Записати в термінал: git remote add origin https://github.com/vladio0/MyFirstProject.git
-> git push -u origin main


Як запустити проект:

натиснути кнопку "<>Code"

копіювати посилання
https://github.com/vladio0/MyFirstProject.git

за допомогою терміналу відкрити проект 
git clone https://github.com/vladio0/MyFirstProject.git
